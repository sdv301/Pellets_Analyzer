# Pellets Analyzer — Интеллектуальный анализ топливных пеллет

[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)
[![Flask 3.1.2](https://img.shields.io/badge/flask-3.1.2-lightgrey.svg)](https://flask.palletsprojects.com/)
[![Scikit-Learn](https://img.shields.io/badge/scikit--learn-1.7.2-orange.svg)](https://scikit-learn.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Pellets Analyzer** — это современная аналитическая платформа на базе Flask, предназначенная для исследования, визуализации и оптимизации характеристик топливных пеллет. Система объединяет управление экспериментальными данными, продвинутый ML-прогнозист и модуль ИИ-аналитики.

---

## ✨ Основные возможности

### 📊 Продвинутая визуализация
- **Интерактивные графики**: Использование Plotly для создания динамических диаграмм (3D scatter plots, анимированные тренды).
- **Статические отчеты**: Профессиональные графики на основе Matplotlib и Seaborn.
- **Сравнение составов**: Детальное сопоставление до 10 параметров для 5 составов одновременно.
- **Тепловые карты**: Анализ корреляций между компонентами и свойствами пеллет.

### 🤖 Машинное обучение и Оптимизация
- **Hybrid Prediction Engine**: Предсказание свойств на основе Random Forest и Gradient Boosting. 
- **Стратегия Cold-Start**: Автоматический переход на линейную агрегацию свойств компонентов, если ML-модели еще не обучены на достаточном объеме данных.
- **Constrained Optimization**: Поиск оптимального состава пеллет по заданным критериям (максимизация теплоты сгорания, минимизация зольности) с учетом ограничений по доступности компонентов.
- **Feature Importance**: Анализ влияния каждого компонента на итоговые характеристики продукта.

### 🔍 ИИ-Аналитика
- **Контекстный анализ**: Обработка запросов пользователя на естественном языке для выявления скрытых трендов.
- **Система рекомендаций**: Автоматическая генерация советов по улучшению рецептуры пеллет на основе накопленной базы знаний.

### 📁 Управление данными
- **Умный импорт**: Автоматическая нормализация колонок при загрузке из CSV и Excel (распознавание различных форматов названий и единиц измерения).
- **SQLite Database**: Надежное хранение измеренных параметров, составов компонентов и истории оптимизаций.
- **Экспорт отчетов**: Сохранение результатов анализа в PDF и Excel.

## 🛠 Технологический стек

- **Backend**: Python 3.8+, Flask 3.1.2, Flask-Session.
- **Data Science**: Pandas, NumPy, Scikit-learn (ML), SciPy (Optimization).
- **Visualization**: Plotly, Matplotlib, Seaborn.
- **Database**: SQLite3.
- **Frontend**: HTML5, Vanilla CSS, JavaScript, Jinja2.

## 🚀 Быстрый старт

### Требования
- Python 3.8 или более поздняя версия.
- Установленный `pip`.

### Установка

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/sdv301/Pellets_Analyzer.git
   cd Pellets_Analyzer
   ```

2. **Создайте и активируйте виртуальное окружение:**
   ```bash
   # Windows
   python -m venv venv
   venv\Scripts\activate

   # Linux / macOS
   python3 -m venv venv
   source venv/bin/activate
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Запустите приложение:**
   ```bash
   python main.py
   ```

После запуска приложение будет доступно по адресу: `http://127.0.0.1:5000/`

## 🏗 Архитектура проекта

- `main.py` — Главный модуль, управление маршрутизацией и сессиями.
- `ml_optimizer.py` — "Сердце" системы: ML-модели, парсеры составов и логика оптимизации.
- `data_processor.py` — Обработка входящих файлов, проверка данных и нормализация.
- `database.py` — Слой доступа к данным (DAO) для работы с SQLite.
- `gui.py` — Модуль генерации всех типов графиков и визуальных отчетов.
- `ai_ml_integration.py` — Высокоуровневая интеграция ML-инсайтов и ИИ-аналитики.
- `templates/` — HTML-шаблоны страниц.
- `static/` — CSS-стили, JS-скрипты и сгенерированные изображения.

## 📋 Подготовка данных для загрузки

Для корректной работы системы файлы Excel/CSV должны содержать:
1. **Состав (Composition)**: перечисление компонентов в процентах (например: *"Опилки 70%, Торф 30%"*).
2. **Параметры**: Плотность, Теплота сгорания (q), Зольность (Ad) и другие измеряемые характеристики.
3. **Лист "Компоненты"** (для Excel): должен содержать базовые характеристики отдельных видов сырья.
